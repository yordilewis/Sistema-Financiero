# Descripción

**LondX** es un sistema de gestión de préstamos y cobros desarrollado como proyecto final de la asignatura **Programación III** de la Universidad Abierta para Adultos (UAPA).

El sistema permite administrar de forma organizada la información de clientes, préstamos, cuotas, pagos y usuarios, facilitando el control de las operaciones financieras mediante una interfaz intuitiva y una base de datos estructurada.

---

#  Objetivo

Desarrollar una solución informática que permita administrar de manera eficiente los procesos relacionados con la gestión de préstamos y cobros, aplicando los principios de programación orientada a objetos, bases de datos y buenas prácticas de desarrollo de software.

---

#  Funcionalidades Principales

- Inicio de sesión.
- Gestión de usuarios.
- Gestión de clientes.
- Gestión de préstamos.
- Gestión de cuotas.
- Registro de pagos.
- Dashboard.
- Reportes.
- Auditoría.

#  Instalación y Ejecución

1. Clonar el repositorio.

```bash
 git clone https://github.com/yordilewis/Sistema-Financiero.git
```

2. Abrir el proyecto en **Visual Studio 2022**.

3. Restaurar la base de datos utilizando los scripts SQL incluidos.

4. Configurar la cadena de conexión a SQL Server.

5. Ejecutar el Backend.

6. Ejecutar el Frontend.

7. Iniciar sesión en el sistema.

# 🗄 Base de Datos

La base de datos fue desarrollada en **SQL Server** y está compuesta por las siguientes tablas principales:

- Roles
- Usuarios
- Clientes
- Prestamos
- Cuotas
- Pagos
- Auditoria

Además, incluye procedimientos almacenados para optimizar las operaciones del sistema.
